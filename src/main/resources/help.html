Summary

TypeTalk provides easy access to MaryTTS.  It includes the MaryTTS backend and several voices.  At this moment only English voices are included.  TypeTalk can be started with a minimized window in order to let you load it at login.

Using the UI

TypeTalk has two areas for typing.  A single-row bottom text field and a top area meant for larger text input . When the user interface first opens the single-row text field grabs focus and allows you can begin typing.  When you hit the enter key the text is moved to the top text area and TypeTalk speaks the words you typed.  In this way using the interface is similar to using a messaging application.



The top text area is meant for larger portions of text. When you hit the 'play' button on the bottom right the text is spoken to you. With the 'save' button you can save the output to a wave file. When TypeTalk encounters a word that the MaryTTS back end cannot pronounce it shows a little popup at the top-right corner of your screen with the unpronounceable word. You can correct the text in the top text area to enable a complete message.

Configure voice

You can configure the by clicking 'Configure' - >'Configure voice'. All configuration options in the dialog show a little help button with a explanation about the effect. You can listen to the effect through a text field and play button at the bottom of the dialog.

Configure the UI

You can configure the behavior of the ui by clicking 'Configure' -> 'Configure graphical interface'. Through this dialog you can disable the splashscreen, the welcome screen at start up and whether the ui should be started minimized or not. In this dialog you can also change the keyboard shortcut for toggling the visibility of the ui.
